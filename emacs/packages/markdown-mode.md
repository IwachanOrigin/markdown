
# markdown-mode

## 1. Create Table

    1, まとまったデータをcsv形式などで作成します。  
    * 例  
    No,Title
    1,A
    2,B
    3,C

    2. まとまったデータを選択し、「C-c C-c |」を入力すると、以下のような表が出来上がります。  

| No | Title |
| 1  | A     |
| 2  | B     |
| 3  |       |

    何故か末尾の文字列が消えてしまいます。私の環境依存かもしれません。  
    消えても良い任意の文字を末尾に追加しておくと任意文字が変わりに消えてくれます。  

    3. 以下表の様に「|-|」を入力します。  

| No | Title |
|-|
| 1  | A     |
| 2  | B     |
| 3  | C     |

    4. 「|-|」のエリアで「C-i」すると、ヘッダーもいい感じに調整してくれます。  

| No | Title |
|----|-------|
| 1  | A     |
| 2  | B     |
| 3  | C     |
