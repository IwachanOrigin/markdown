* 概要
MSYS2上のemacsを、MSYS2のシェル経由で起動する方法。

* 手順
** バッチ作成
1) 以下のようなコマンドプロンプトで動くバッチファイルを作成する

   #+begin_src shell
     @echo off
     set "MSYS2=C:\software\msys2_64"
     %MSYS2%\msys2_shell.cmd -defterm -here -no-start -ucrt64 -shell bash -lc "runemacs"
   #+end_src

2) 上記バッチファイルを実行する。
